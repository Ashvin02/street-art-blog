# Adapter un site aux différents supports : Exercice de retrofitting

## Sujet/Brief
Intégrer une page web **« Street art blog »** en **responsive web design**, en utilisant le principe du **retrofitting**.

---

## Contexte
Vous allez appliquer vos connaissances en intégration web pour rendre une page responsive, en commençant par la version **desktop**.

---

## Consignes

1. **Analyse de la structure**
   - Analyser la structure de la page (utilisez une feuille de papier et un crayon si nécessaire).

2. **Intégration HTML**
   - Intégrer la partie HTML en respectant la **sémantique** et la **hiérarchie des titres**.
   - Valider le code HTML avec le [validateur W3C](https://validator.w3.org/).

3. **Application des CSS**
   - Appliquer les CSS pour la **version desktop** en valeurs fixes.
   - Découvrir et appliquer les techniques pour :
     - Gérer les **images adaptatives** aux différents supports.
     - Utiliser des **tailles fluides** (et non fixes) pour les polices et les blocs.
     - Appliquer des **mediaqueries** pour adapter l’affichage aux versions **tablette** et **mobile**.

---

## Informations utiles pour l’intégration

| Élément                     | Détail                                                                 |
|-----------------------------|------------------------------------------------------------------------|
| **Largeur du contenu**      | 960px (centré, fonds et image du header en pleine largeur).            |
| **Image du header**         | Ne pas agrandir au-delà de sa taille réelle.                          |
| **Effet au survol**         | Bordure sous le texte des rubriques du menu.                          |
| **Police par défaut**       | Source Sans Pro (Google Fonts), couleur `#777777`.                     |
| **Pictogrammes**            | Font Awesome pour les blocs textes et réseaux sociaux.                 |

### Tailles de police (valeurs fixes)
| Élément               | Taille  |
|-----------------------|---------|
| Texte par défaut      | 16px    |
| Menu                  | 24px    |
| Titre principal       | 56px    |
| Sous-titre            | 32px    |
| Citation              | 30px    |
| Titres des blocs      | 28px    |
| Copyright             | 12px    |

---

## Contraintes
- Utiliser le principe du **retrofitting** (intégration de la version desktop en premier).
